日本大学院考修士-机器人和人工智能怎么选

发表于 2019-04-26 | 分类于 日本理工科大学院入试说明及备考攻略

从国内的专业设置上来讲，机械设计制造及其自动化更偏向于日本的机器人方向，因为机器人本身也是典型的机电一体化系统。而人工智能方向较多集中在计算机系和数学系。从这点来看，机器人专业和人工智能专业有着较大的区别。在日本，研究机器人的老师集中于工学系（机械工学，电气电子，精密工学），研究人工智能的老师集中于情报理工和理学系（数理科学等）。

但是，机器人专业与人工智能专业又有着紧密的联系。一般来讲，人工智能支持了机器人强大的计算能力（类比人类大脑），而机电系统构成了机器人的执行机构（类比人类四肢）。大脑和四肢的协调能力使得人类可以完成复杂的工作。所以在日本的很多大学，两个学科有着紧密的交叉，专业名称通常为“知能システム”“知能機械”等。

从日本大学院修士一般入试角度来看，总的来讲，与机器人专业相比，人工智能专业对数学能力的考察更加重视（题型较多较难）。机器人专业的数学考试通常涉及微积分，线性代数和概率论等，人工智能专业还会涉及复变函数，离散数学等。专业课方面，机器人专业更要注重考察机械力学，控制理论，电路电子等，人工智能专业更注重算法与数据结构，编程语言等。

日本理工科大学院G30/SGU英文项目常见面试问题汇总

发表于 2019-04-24 | 分类于 日本理工科大学院SGU英文项目申请攻略

前言：由于G30/SGU英文项目的筛选通常没有笔试环节，如果进入了面试环节，那就说明学生的条件是基本达到了该学校的要求。因此面试是最后一个重要环节。学生需深刻理解面试的根本目的，以便更加沉着冷静的应对面试：

  1. 考察学生的语言能力和沟通能力。尤其是对于国际项目的学生，学校及教授要考察学生能不能用英语进行有效的生活及研究方面的沟通交流。

  2. 考察学生对于自己过去研究的理解，尤其是现在中介泛滥，学校及老板要确认学生本人对于毕业论文及研究计划的理解及真实性。

  3. 考察学生的逻辑思维能力及现场反应能力。这一条要求学生在进行充分的准备之后，面试时能够自信沉着冷静的应对，无需过分紧张。切记，语速不宜过快。即使是东京大学这样的学校，也可能遇到日式英文口音极重的老板，此时不用紧张，听不懂时可以大方有礼貌得请老师重复问题。如果还听不懂，就按照自己的理解说一说，说完以后问老师，请问这是您想问的问题吗？多数情况下，老板不会追问。

  4. 日本人很讲究着装，因此要注意着正装或上身正式衬衫。头发不能过分夸张，以简洁为主。面带微笑，落落大方，不仅要自信，而且要谦虚，表现出上进好学的态度。

以下是作者通过在东京大学研究教学期间，与合作过的一些老板交流后的经验，以及帮助过的一些学生的面试经验而总结出的一些面试常见问题，希望能认真准备，有助于在面试中能够沉着冷静的应对各种问题。（如果在面试中遇到已经准备好的问题，也要先思考2秒，再慢慢回答。如果遇到没有准备到的问题，请认真思考，有条理得回答，一般性问题没有对错，不要太激进即可，只考察表达能力。）

1 一般性问题 （这里以东京大学为例）

1.1 自我介绍（简短有力，不宜过长）。准备一个自己的优势（可以说自己做事认真，因为日本人喜欢认真努力的人。或学习适应能力强）和一个劣势（劣势可以说日语水平一般）

1.2 为什么不在原来学校继续读。

1.3 为什么要选择日本（喜欢日本哪些东西，即自己的兴趣爱好），为什么选择东京大学，东京大学和你原来学校相比有什么异同。（与1.2是同一个性质问题）

1.4 为什么选择这个专业，为什么选择这个研究室，请谈谈你对选择研究室研究方向的理解。如果专业有不同，那为什么要换方向，你觉得在东京大学选择的方向和原来的专业有什么异同点。

1.5 经济状况，如果没有奖学金怎么办。（回答方向：不打工，父母支持自己）

1.6 是否还报考了其他学校及专业。（可以选择性实话实说）

1.7 毕业之后的打算。（有些老师会比较倾向于考虑读博士，打算专心做科研的人）。有没有打算毕业之后去美国等其他国家。

1.8 有没有打算学日语。（在不影响科研的前提下，认真学习日语）

1.9 如果这次没有考上怎么办。

2 专业性问题

2.1 本科期间有没有做过什么专业相关的项目及课题，有哪些收获，如果有发表论文，请准备好。

2.2 毕业论文概括介绍。请充分熟悉自己的毕业论文，准备一个简短的介绍，需要涵盖研究目的，研究方法，结果，及将来计划。如果遇到专业十分相关的教授，可能会问详细问题。

2.3 研究计划简单介绍。（研究目的及浅谈对该领域的了解，需要提前看对应老板的文章，一定要对老板的核心研究内容有准确的把握和理解）

日本理工科大学院G30/SGU英文项目概述

发表于 2019-04-23 | 分类于 日本理工科大学院SGU英文项目申请攻略

什么是G30/SGU英文项目？

一句话概括，G30/SGU是日本的英文授课项目。

申请过程与申请欧美大学类似，需要托福或者雅思成绩，不需要掌握日语。

G30/SGU项目由来

2009年，日本政府为提高大学国际影响力，吸纳全球优秀学生开展了Global30英语项目，简称G30。项目选取了七所国立大学：東京大学，京都大学，大阪大学，東北大学，筑波大学，名古屋大学，九州大学；以及六所私立大学：慶應義塾大学，早稲田大学，立命館大学，上智大学，明治大学，同志社大学。

截止至2014年的G30项目，已经接纳来自全世界成千上万的的优秀学子赴日深造学习。但是日本政府一直没有对G30项目进行扩军，而是在2014年开展了全新的“Super Global Univercity Project”,简称SGU英文项目，院校也由G30的13所扩充至前面提到的SGU37所，英语项目的开设主要是为了提升日本大学在国际上的排名，展示优秀的教育资源和科研环境，并且在全球化竞争中抢夺人才。

为什么选择G30/SGU英文项目？

首先是申请时无需日语成绩，而且可以在海外直接完成申请与录取过程。与欧美国家相比，对雅思和托福成绩的要求更低，留学成本也较低，且入学后的奖学金比例较高，同时日本的教育与学术水平在世界也是顶尖的。该项目覆盖代表日本最高水平的研究领域（计算机，情报，机械，电气电子，生命科学等）。最后，掌握中英日多种语言，有利于提高国际化的就职能力。

为什么各个大学网站上面缺少G30/SGU相关信息？

G30/SGU 项目承诺全部课程为英文授课，但是SGU大学里面有的大学院并不能确保所有课程为英文授课，甚至有的大学院某些专攻能够全英文授课，某些专攻则不能，因此从全校层面很难汇总英文项目的信息。同时由于在日本各个院系的权利很大，因此G30/SGU项目更多的是以学院为基本单位进行汇总的，所以全英授课项目通常要在各个学院的主页上进行查询。

日本理工科大学院热门专业及教授介绍（三）-机器人学（Robotics）

日本理工科大学院热门专业及教授介绍（三）:机器人学（Robotics）

机器人学【robotics】是与机器人设计、制造和应用相关的科学。又称为机器人技术或机器人工程学，主要研究机器人的控制与被处理物体之间的相互关系。全世界已有近百万台机器人在运行,机器人技术已形成为一个很有发展前景的行业。

对机器人行为的描述，以科幻小说家艾西莫夫在小说《我，机器人》中所订立的“机器人三定律”最为著名。其为机器人提出的三条「定律」，程序上规定所有机器人必须遵守：

第一法则：机器人不得伤害人类，且确保人类不受伤害；

第二法则：在不违背第一法则的前提下，机器人必須服从人类的命令；

第三法则：在不违背第一及第二法则的前提下，机器人必须保护自己。

在现实中，“三定律”成为机械伦理学的基础。

从分类上，机器人主要分为以下几大类：

工业机器人，

人形机器人，

医疗机器人，

特种机器人，

以及服务机器人。

日本理工科大学院热门专业及教授介绍（二）:人工智能/机器学习

人工智能的浪潮正在席卷全球，诸多词汇时刻萦绕在我们耳边，例如：人工智能（Artificial Intelligence）、机器学习（Machine Learning）、深度学习（Deep Learning）。总的来说，人工智能是研究、开发用于模拟、延伸和扩展人的智能的理论、方法、技术及应用系统的一门技术科学。“人工智能”是“一门技术科学”，它研究与开发的对象是“理论、技术及应用系统”，研究的目的是为了“模拟、延伸和扩展人的智能”。而机器学习是一种实现人工智能的方法，深度学习是一种实现机器学习的技术。

本文重点介绍日本大学院人工智能/机器学习方向的热门学校及教授，同时，针对每位教授，根据教授的近年来课题基金方向，本文给出了联系教授时的研究计划书方向，以及申请者能够作为加分项的相关能力。

日本理工科大学院热门专业及教授介绍（一）:车辆工程

通常来讲日本是没有车辆工程专业的，大多数的车辆方面老师集中在机械工程，电气工程，以及情报理工，一部分做交通相关的老师会在土木工程。不过也有例外，比如九州大学就有专门的automotive science学科。由于车辆专业在日本的复杂性，作为日本理工科大学院热门专业及教授介绍的开篇，我们来介绍车辆工程专业。

首先，日本工科比较有名的大学都会有做车辆方面研究的教授，但是车辆是一个极度交叉学科，尤其随着近期自动驾驶的火热发展，各个学校各个研究室研究的方向也是五花八门，各有所长。其次，日本的车辆毕业就业情况很好，Nissan，Toyota，Mazda都是不错的选择，以及专门做自动驾驶的TierIV，ZMP等。综上，在日本读修士或博士选择车辆方面的导师将会是不错的选择。
